Let's do some interactive coding today!

On Chrome and Firefox, to open the Browser Console, right-click the page and select 'Inspect'. The console will appear at the bottom or on the right side of the screen. On Opera it will be 'Inspect Element'

If you're ready for a shortcut, press 'CMD+OPT+J' on Mac, or 'F12' on Windows

On Safari, go to the menu bar and open 'Safari > Preferences > Advanced >' and tick the box 'Show Develop Menu', restart Safari, now you can right-click on the page and select 'Inspect Element' to see the console.
